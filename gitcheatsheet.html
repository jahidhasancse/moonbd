How to push my code on Github through git bash/ os bash?

Change directory (Forward): cd 'fileName'

Change directory (Backward): cd..

Find files (OS): dir

Bash clear(OS): cls

Bash clear(git): clear

Git Version: git --version

Find files (git): ls

Repo initialized: git init

Add Readme file: git add README.md

Add my files: git add .

Add commit: git commit -m "My first git commit"

Identify Branch: git branch -M main

Add Url: git remote add origin https://github.com/****

Code Push: git push -u origin main

Happy Coding...!!!
